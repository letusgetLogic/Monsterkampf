# Monsterkampf-Simulation
Ein Konsolenspiel mit **ASCII-Zeichen** und der Programmiersprache **C#**.

------------------------------------------------------------------------

## Beschreibung
Der Kampf zwischen 2 Monstern soll realisiert werden. Die Monster haben vorgegebene Rassen wie Ork, Troll und Goblin. Ihre Werte für die folgende Attributen müssen vom Benutzer angegeben werden: Lebenspunkte (HP), Angriffsstärke (AP), Abwehrpunkte (DP), Geschwindigkeit (S)

## Projekt
Dieses Projekt wurde für das Modul **4FSC0PD001** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **28.10.2024 - 22.11.2024**
