# Monsterkampf-Simulation
Ein Konsolenspiel mit **ASCII-Zeichen** und der Programmiersprache **C#**.

------------------------------------------------------------------------

## Beschreibung
Der Kampf zwischen 2 Monstern soll realisiert werden. Zwischen den Rassen wie Ork, Troll und Goblin müssen 2 unterschiedliche gewählt werden. Ihre Werte für die folgende Attributen müssen vom Benutzer angegeben werden: Lebenspunkte (HP), Angriffsstärke (AP), Abwehrpunkte (DP), Geschwindigkeit (S).

## Projekt
Dieses Projekt wurde für das Modul **4FSC0PD001** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **28.10.2024 - 22.11.2024**

Die Aufgaben waren
- Erstellung von Zeitplan mit Meilensteinen,
- Erstellung von Vererbungs-Klassen,
- Initialisieren von Werten durch Eingabe,
- Berechnung von Kampfablauf,
- Ergebnisse über den Sieger und die Anzahl von Runden.

## Zusätzliche Features
- Wette-Eingabe vor Kampf für Vergleich mit dem Ergebnis
- 2 Fähigkeiten pro Monster
- Visualisierung von Anzeige-Balken von Attributen, zB. HP 80/100  |////////__| 
- Kampfanalyse mit ASCII-Diagrammen

------------------------------------------------------------------------

## Spiel
Um das Spiel auszuführen, müssen folgende Schritte gemacht werden:
1. Downloade den Ordner "Monsterkampf-Build.zip"
2. Extrahiere den Ordner
3. Wähle .exe-Datei aus
4. Lass den Kampf beginnen!
